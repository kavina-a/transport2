# Smart Public Transport — Parliamentary Sub-Committee Submission

Professional, government-facing slideshow (11 slides) for a Sri Lankan parliamentary audience. Submission to the Parliamentary Sub-Committee on Ancillary Business Models — Transport, chaired by Deputy Minister Chathuranga Abeysinghe.

**Presenter:** Rasal Jayasinghe  
**Focus:** Ancillary business models for transport; pilot = public buses.  
**Core idea:** Technology-driven, efficiency-first public transport using smart bus stands and transponder-based live bus tracking. No booking or payment in phase one.

---

## How to present

1. **Open the presentation:** Open `index.html` in a modern web browser (Chrome, Firefox, Safari, Edge).
2. **Full screen:** Press `F` for full-screen presentation mode (or use the browser’s full-screen option).
3. **Speaker notes:** Press `S` to open the speaker notes window in a separate window (notes are also in `SPEAKER-NOTES.md`).
4. **Navigation:** Use arrow keys or space to move between slides; `Esc` shows slide overview.
